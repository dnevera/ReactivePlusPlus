\subsubsection*{template$<$rpp\+::constraint\+::decayed\+\_\+type TDisposable$>$\newline
class rpp\+::disposable\+\_\+wrapper\+\_\+impl$<$ TDisposable $>$}
Wrapper to keep disposable. Any disposable have to be created right from this wrapper with help of {\ttfamily make} function. 

Member functions is safe to call even if internal disposable is gone. Also it provides access to "{}raw"{} shared\+\_\+ptr and it can be nullptr in case of disposable empty/ptr gone.

Can keep weak\+\_\+ptr in case of not owning disposable
